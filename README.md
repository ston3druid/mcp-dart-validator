## Description

This project provides simplified Dart validation tools by leveraging the built-in `dart analyze` command. The codebase has been streamlined from ~4000 lines to ~200 lines for better maintainability and performance.

## Features

- Fast Dart code validation using `dart analyze`
- MCP (Model Context Protocol) integration
- Simplified API for easy integration
- Minimal dependencies and footprint
